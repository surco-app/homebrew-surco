# Homebrew Surco

Homebrew tap para [Surco](https://github.com/vigosan/surco-releases) — organizador de pistas de audio para DJ.

## Instalación

```bash
brew install --cask vigosan/surco/surco
```

Tras instalar, la app se actualiza sola mediante su auto-updater integrado
(`auto_updates true`), por lo que `brew upgrade` no la gestiona.

## Desinstalación

```bash
brew uninstall --cask surco
brew uninstall --zap --cask surco   # elimina también datos de usuario
```
